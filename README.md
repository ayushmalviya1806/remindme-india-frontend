## Overview

Frontend for [RemindMe India](https://remindmeindia.site) — India's first AI-powered WhatsApp reminder assistant. Serves as the marketing landing page and Pro subscription checkout flow.

Built with AI-assisted development using Claude and Windsurf IDE + Cascade AI.

**Live at:** [remindmeindia.site](https://remindmeindia.site)

**Backend repo:** [ayushmalviya1806/remindme-india](https://github.com/ayushmalviya1806/remindme-india)

---

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| React | UI framework |
| Tailwind CSS | Styling |
| Razorpay Checkout.js | Payment processing |
| Vercel | Hosting & deployment |
| Hostinger | DNS management |

---

## Features

- **Landing page** — Product overview, features, pricing, and CTA to WhatsApp
- **Pro checkout page (`/pro`)** — Razorpay Checkout.js integration for ₹99/month Pro plan
- **Auto user creation** — Users who pay on the website are automatically created in the backend
- **Responsive design** — Mobile-first, optimized for Indian users on phones
- **WhatsApp deep links** — One-tap to start chatting with the bot

## Deployment

Deployed on **Vercel** with automatic deploys from the `main` branch. Custom domain `remindmeindia.site` configured via Hostinger DNS.
